# Pokedex<br>
PokeDex는 사용자가 포켓몬을 검색하고 자세한 정보를 확인할 수 있는 웹 애플리케이션입니다.  
포켓몬의 타입, 설명 등을 확인할 수 있으며, 사용자 친화적인 UI를 제공합니다.<br>
